About Weather Now

Weather Now is a modern web-based weather information platform designed to deliver real-time weather insights in a simple, reliable, and user-friendly manner.

Who We Are

Weather Now is a front-end weather forecasting project developed as part of a skill development activity. The main objective of this project is to apply web technologies such as HTML, CSS, and JavaScript to build a real-world, API-driven application.

The platform fetches live data from the OpenWeatherMap API and presents it through a clean, responsive dashboard suitable for everyday users.
